# Main Menu

The main menu consists of THREE primary screens navigated horizontally (D-Pad left/right, L/R):

They are `Multi Play`, `Single Play`, and `Commissioner`.

## Multi Play

Choose from a variety of exhibition style matches for 1 to 4 players.  The `Multi Play` screen consists of the following options:

- Exhibition
- Royal Rumble
- Pay-Per-View
- King of the Ring
- Guest Referee
- Ladder Match
- Ironman Match

### Exhibition

From the manual:

> Select a game mode, then select which controllers will be used or select WATCH to view a CPU vs. CPU match. Now choose from the 10 arenas to compete in and the rules for the match (see "Selecting the Rules" on page 32). Finally, determine which belt will be on the line and press the A Button.

> Select a Superstar by pressing the Control Pad, then press the A Button to confirm. To select from a different stable of Superstars, press the L or R Button.

> If you've made a mistake, you can press the B Button at any time to return to the previous menu.

When selecting `Exhibition`, there are five pages to go through before selecting the characters to have a match.

- Match
- Player
- Arena
- Rules
- Belt

#### Match

On this page, the player selects the kind of match.  The options are:

- Single Match - A one vs. one match
- Tag Match - Two vs. two
- Triple Threat Match - One vs. one vs. one 
- Handicap Match - Two vs. one
- Cage Match - One vs. one in a cage match, where the person who climbs out of the cage wins

#### Player

On this screen, the player selects which character is controlled by which controller, or assigns characters to the CPU.  The `Player` screen shows different options depending on which match type was selected on the previous screen, as well as how many game-pads are connected.  The game allows up to 4 players.  The options for each are as follows:

		- CPU & CPU vs. 1P & 2P
		- 1P & CPU vs. 2P & CPU
		- 1P & CPU vs. CPU & CPU
		- CPU & CPU vs. 1P & CPU
		- Watch (CPU & CPU vs. CPU & CPU)

- Triple Threat Match

	- 1P vs. 2P vs. 3P
	- 1P vs. 2P vs. CPU

## Single Play

Play through the main 1-player game modes to win belts and to reveal hidden Superstars.  The `Single Play` screen consists of the following options:

- Championship
- Survival

### Championship

Career paths allowing you to compete for the following championship belts, earning in-game money which can be used to unlock items in the `Smackdown Mall` shop:

- World Heavyweight - see `https://wwfnomercy.fandom.com/wiki/World_Heavyweight_Championship`
- Intercontinental - see `https://wwfnomercy.fandom.com/wiki/Intercontinental_Championship`
- Tag Team - see `https://wwfnomercy.fandom.com/wiki/Tag_Team_Championship`
- Women's - see `https://wwfnomercy.fandom.com/wiki/Women%27s_Championship`
- European - see `https://wwfnomercy.fandom.com/wiki/European_Championship`
- Hardcore - see `https://wwfnomercy.fandom.com/wiki/Hardcore_Championship`

### Survival

Set up exactly like `Royal Rumble` in `Multi Play`, but with only 1P against the CPU.  Player is allowed to select manner of elimination of opponents.  Starts with 4 characters in the ring, 1P vs. CPU vs. CPU vs. CPU.  Each elimination earns the player in-game money to be used in the `Smackdown Mall` shop.

#### Superstar Options

`Superstar Options` refers to options related to game characters, which are referred to in-game occasionally as `Superstars`.  The following five options are on the `Superstar Options` page:

- Edit a Superstar
- Create a Superstar
- Clone a Superstar
- Change Stable
- Stables Name

##### Edit a Superstar

Selecting this option takes you to the `Superstar Select` menu.  Selecting a `Superstar` will open the [[character-options]] for that Superstar.

##### Create a Superstar

Selecting this option takes you to a variation of the `Superstar Select` menu, called `Save Data Select`, which shows two pages, each a 3x3 grid of nine slots to create a new Superstar.  Existing created characters are also on this page, and selecting a previously created Superstar opens a dialog with the text, "Overwrite this Superstar?", and the ability to select Yes or No.  Once a slot has been selected, the [[character-options]] open for a new superstar.

##### Clone a Superstar

Selecting this option takes you to the `Superstar Select` menu, with the ability to copy a character from one slot to another.
