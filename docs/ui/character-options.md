# Character Options

All game `character slots` have four `attire slots`.  All four must share the same configurations for `Moves`, `Fighting Style`, `Parameter`, and `Ally / Enemy`, but `Profile / Music` and `Appearence` configurations are customizable and allow for signifigant differences between the attire slots.  Additionally, multiple characters may be placed in the `attire slots` for one `character slot`, though they will have the same configurations for `Moves`, `Fighting Style`, `Parameter`, and `Ally / Enemy`. 

Game characters have the following six pages of settings:

1. `Profile / Music` - On this page, the character's `Name` is set along with other such identifying characteristics as `Short Name`, and `Alias`, along with their `Height` and `Weight`.  This is also where their entrance `Music` and their `Titantron` entrance video is set.
2. `Appearance` - On this page, all appearance attributes besides the `Height` and `Weight` are set.
3. `Moves` - On this page, `Moves` are assigned to each of the move slots for the character.
4. `Fighting Style` - Various options are set relating to how the character acts and what they do.
5. `Parameter` -  On this page, offensive and defensive strength values are set for `Head`, `Body`, `Arms`, `Legs`, and `Flying`.
6. `Ally / Enemy`

## 1. `Profile / Music`

In this page, the following are set:

1. `Name` - This is the name for the character.
2. `Short Name` - This is a shorter version of the character's name, used in places where the `Name` would not fit.
3. `Alias` - If set, this is the name shown on the in-match `Attitude Meter`.  If left blank, the `Short Name` is used.
4. `Picture` - A 64px x 64px png image with the character's face and a transparent background.
5. `Height` - Each character is assigned a `Height` between 5'0 and 7'11
6. `Weight` - Each character is assigned a `Weight` between 100 lbs and 599 lbs.
7. `Music` - The song which will play when a character is making their entrance.
8. `Titantron` - A short entrance video which plays at the beginning of a character's entrance.  If left empty, no video will play.

## 5. `Parameter`

Each wrestler has five offensive and five defensive parameter values covering `Head`, `Body`, `Arms`, `Legs`, and `Flying`. 

Each value ranges from 1 to 5, and all ten values combined may not exceed 30 points. These parameters determine how much damage a wrestler deals with moves targeting each body region, and how much damage they receive from attacks to those same regions. Created wrestlers begin with all values set to 1 (10 points total) and may distribute up to 20 additional points as desired.

Offensive and defensive strength values are set for `Head`, `Body`, `Arms`, `Legs`, and `Flying`.   Each is assigned a value between `1` and `5`,  with the total allotment of points being `30`.
