# Superstar Select

The **Superstar Select** screen is the interface used any time WWF No Mercy requires the player to choose a character—whether for Exhibition matches, Championship mode, Multiplayer, or Edit Mode. It presents a grid‑based roster browser, a 3D character preview, and controls for navigating between roster pages and attire slots.

## Overview

Upon loading, the Superstar Select screen displays:

- A **3×3 grid** of character avatar portraits
- A **stable name** (page title) above the grid
- A **3D character model** on the right side
- Character **name, height, weight, and Type** (attire slot) at the bottom

Only slots containing characters are shown. Empty slots do not appear, and empty pages are skipped entirely.

## Grid Navigation

### Movement

- The **D‑Pad** moves the cursor between the visible character slots in the 3×3 grid.
- The currently highlighted slot is shown with a **thicker border**.

### Page Switching

The roster is divided into multiple “stable” pages, each containing up to nine characters.

You can move between pages in two ways:

- **D‑Pad Left** from the leftmost slot moves to the previous page
- **D‑Pad Right** from the rightmost slot moves to the next page
- **L Button**: instantly move one page left
- **R Button**: instantly move one page right

There are **ten total pages** available, but only pages containing at least one character slot will appear.

## Attire Slot Selection

Each wrestler has multiple attire slots, referred to as **Types** (Type 1, Type 2, etc.).

- **C‑Left** and **C‑Right** cycle between a character’s available attire slots.
- The currently selected attire is displayed as **Type X** in the character info panel.

## Character Model Controls

The 3D model on the right side of the screen can be manipulated for inspection:

- **Control Stick Left/Right**: rotate the model to view front, sides, and back
- **Control Stick Up**: zoom out (model becomes smaller)
- **Control Stick Down**: zoom in (model becomes larger)

This allows players to preview attire details, accessories, and body type.

## Character Information Display

At the bottom of the screen, the following information is shown for the highlighted character:

- **Name**
- **Height**
- **Weight**
- **Type** (attire slot currently selected)

These values update instantly as the player moves between characters or changes attire slots.

## Stable Pages (Roster Pages)

A brand‑new save file contains the following stable pages:

1. **Superstars 1**
2. **Superstars 2**
3. **Superstars 3**
4. **Superstars 4**
5. **Superstars 5**
6. **Ladies**

Additional pages appear as the roster expands:

- **Hidden 1** and **Hidden 2**

    - Unlocked by completing in‑game objectives

- **Edit 1** and **Edit 2**

    - Automatically used for newly created CAWs

Pages only appear if they contain at least one character slot.

## Change Stable (Reorganizing the Roster)

The **Change Stable** feature allows any character—default, hidden, or CAW—to be moved to a different slot on any page. This enables:

- Custom roster organization
- Grouping characters by faction, era, or personal preference
- Moving CAWs out of Edit pages into Superstars pages

The system is fully flexible: any slot on any page can hold any character.
